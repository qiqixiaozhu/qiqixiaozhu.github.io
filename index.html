HTML表单元素

Veröffentlicht am 2017-09-14

<input>元素

type属性

text 文本框。

定义单行的输入字段，用户可在其中输入文本，默认宽度为20个字符。

Password 密码框。

定义密码字段，该字段中的字符被掩码（显示为星号或原点）。

hidden 定义隐藏的输入字段（隐藏字段对用户不可见）。

radio 定义单选按钮。

checkout 定义复选框。

button 定义可点击按钮。常用于用户点击按钮时启动JavaScript程序。

reset 定义重置按钮。重置按钮会清除表单中的所有数据。

Image 定义图像形式的提交按钮。必须把src属性和alt属性与之结合使用。

submit 定义提交按钮。提交按钮会把表单数据发送到服务器，数据会发送到表单的action属性中指定的页面。

File 定义输入字段和浏览按钮，供文件上传。

value属性

  • 为input元素设定值。规定输入字段的初始值。
  • 另外，和中必须设置value属性。
  • 不能与一同使用。

placeholder属性

  • 提供可描述输入字段预期值的提示信息。
  • 该提示会在输入字段为空时显示，并会在字段获得焦点时消失。
  • 此属性适用于以下类型：text，search，url，telephone，email及password。

readonly属性

  • 属性规定输入字段为只读（不能修改）。
  • readonly属性不需要值，等同于readonly=”readonly”.

disabled属性

  • 规定输入字段是禁用的（被禁用元素不可用、不可点击、不会被提交）。
  • disabled属性不需要值，等同于disabled=”disabled”

size属性

  • 规定输入字段的尺寸（以字符计）

maxlength属性

  • 规定输入字段允许的最大长度。
  • 若设置maxlength属性，则输入控件不会接受超过所允许数的字符。
  • 该属性不会提供任何的反馈信息。若需要提醒用户，需编写JavaScript代码。
